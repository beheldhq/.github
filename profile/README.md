## What is Beheld?

Beheld is a developer-first portfolio platform — built around the work, not the resume. Sync your repositories, deploy a portfolio that updates as you ship, and connect with companies that care about what you actually build.

The way developers present themselves today is broken. Resumes flatten years of craft into bullet points. LinkedIn rewards networking, not engineering. GitHub shows code but loses the story. Beheld closes that gap.

> **Current status: early development.** Public access is rolling out gradually. Watch the repos or follow [@beheldhq](https://x.com/beheldhq) to know when it opens.

## Quick start

```bash
npm install -g @beheld/cli
beheld login
beheld init
```

Full setup guide lives in [docs](https://github.com/beheldhq/docs).

## Built with

- **Web** — React, Vite, TailwindCSS
- **API** — Node.js, TypeScript
- **CLI** — Node.js, distributed via npm
- **Infra** — Cloud-native, deployed serverless

## Contributing

Beheld is early. We're building `cli` and `docs` in the open while the web app and API stabilize. Ways to help:

- Open an issue on [`cli`](https://github.com/beheldhq/cli/issues) — bugs, ideas, requests
- Send a PR to [`docs`](https://github.com/beheldhq/docs) — typos, clarifications, examples
- Share feedback at [hello@beheld.dev](mailto:hello@beheld.dev)
